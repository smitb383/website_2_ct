5 in 5

I did a series of sketches to expand my processing vernacular and challenge myself to actively engage with political news, that I normally actively ignore to prevent being upset. I created a code project using a new processing library everyday that dealt with the subject of the first Trump headline I saw that day. This making practice and set of fixed constraints enabled me to gain fluency not only in the processing programming language but also in the weeks political conversations that I normally find easier to disregard.

1

New Trump Conspiracy Theory Tweeted With No Evidence

Playing with the subject of conspiracy theories and "doctored information" within my daily article, I used the processing image library to create a program that displays an image of the moon-landing and allows the user to adjust and warp a conspiracy theory image . I related the preposterousness of Trump's conspiracy theory claim to that of the moon-landing conspiracy.

2

Al Sharpton criticizes Trump for saying Aretha Franklin worked for him

The subject of my daily article being Trump disrespecting Aretha Franklin, I used the processing recorded video library to create a program that plays a video of Aretha Franklin's performing Respect in 1967. I experimented with user interaction and video so that when the user holds down and drags the mouse horizontally, the video will fast forward or rewind relative to the mouse position on the screen.

3

Donald Trump Isn't the Victim of a Witch-Hunt. He's Leading It

Thinking about a witch-hunt and the nouveau McCarthyism Trump is inspiring, I created a program using the video capture and hyperlink libraries in processing that presents a button to the user that when clicked pulls up the opinion article about trump as a witch-hunter and simultaneously turns on the users computer camera and starts recording them while they are reading the article. The aim was to explore creating censorship in code and emulate the realm a modern day which hunt may operate within.

4

Trump Bump Has Become a Tsunami

Working with the subject tsunami and thinking about making waves in processing, I experimented with the Xyscope library in processing that transforms vector shapes into waveforms and maps then into corresponding audio wavelengths. This allows you to great interactive shapes and images that generate audio in real time.
